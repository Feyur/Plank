# @plank/desktop — десктоп-обёртка Plank (macOS, Tauri)

Нативное окно вокруг веб-приложения Plank. v1 просто открывает прод
(`https://plank.atank.ru:9443`) — вход по cookie работает как в браузере,
потому что окно грузится прямо с этого origin.

## Требования
- Rust (`rustup`) — тулчейн для сборки Tauri.
- Node/npm — уже есть в монорепо.

## Сборка
```
npm run build -w @plank/desktop        # приложение + установщик .dmg → apps/desktop/release/
npm run build:app -w @plank/desktop    # только .app, без установщика
```
Готовые файлы складываются в `apps/desktop/release/` (в git не идёт):
- `Plank_<версия>_aarch64.dmg` — установщик, его отдавать другим;
- `Plank.app` — приложение для запуска напрямую.

Приложение не подписано: при первом запуске из Finder — правый клик → «Открыть»
(или System Settings → Конфиденциальность и безопасность → «Открыть всё равно»).

## Разработка
`npm run dev -w @plank/desktop` открывает окно; правки конфига видны после перезапуска.

## Что где
- `src-tauri/tauri.conf.json` — имя, идентификатор, окно (URL прода), иконки.
- `app-icon.svg` — исходник иконки; пересобрать набор: `npx tauri icon app-icon.svg` (из `apps/desktop`).
